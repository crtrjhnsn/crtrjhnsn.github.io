# Getting Started with Artifact Gallery

Welcome to your personal artifact gallery — a single place to collect, search, and share everything you've built with Claude.

## What is an Artifact?

An **artifact** is any piece of work you want to preserve and revisit:

- **Claude artifacts** — interactive HTML/JS tools generated by Claude
- **Documents** — notes, guides, and markdown files
- **Images** — screenshots, designs, generated visuals
- **Code snippets** — reusable functions, scripts, or configs

## Running the Gallery

The gallery is a static website. Because it fetches files locally, you need a small web server — not just a double-click on `index.html`.

Open a terminal in the `artifact-gallery/` folder and run:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000) in your browser.

## Adding an Artifact

### 1. Drop in your file

Copy your file to the right subfolder:

| Type | Folder |
|------|--------|
| Claude artifact (HTML) | `content/claude/` |
| Document (Markdown) | `content/docs/` |
| Image | `content/images/` |
| Code | `content/code/` |

### 2. Register it in artifacts.json

Open `content/artifacts.json` and add an entry:

```json
{
  "id": "my-unique-id",
  "title": "My New Artifact",
  "description": "A short description.",
  "type": "claude",
  "file": "content/claude/my-artifact.html",
  "tags": ["tag1", "tag2"],
  "date": "2026-04-14"
}
```

> **Tip:** Use the `add-artifact.py` script to add artifacts automatically:
> ```bash
> python3 add-artifact.py
> ```

### Field Reference

| Field | Required | Description |
|-------|----------|-------------|
| `id` | Yes | Unique URL-safe identifier (no spaces) |
| `title` | Yes | Display name |
| `description` | No | Short summary shown on the card |
| `type` | Yes | `claude`, `doc`, `image`, or `code` |
| `file` | Yes | Path to the artifact file |
| `tags` | No | Array of keyword strings |
| `date` | No | ISO date string (`YYYY-MM-DD`) |
| `language` | Code only | Syntax highlight language (e.g. `typescript`) |
| `thumbnail` | No | Path to a preview image for the card |

## Dark Mode

Click the moon icon in the header to toggle dark mode. Your preference is saved locally.

## Deploying

To share your gallery online, upload the entire `artifact-gallery/` folder to any static host:

- **GitHub Pages** — free, push to a repo
- **Netlify** — drag and drop the folder at netlify.com
- **Vercel** — `vercel --prod` in the folder

No backend needed.
